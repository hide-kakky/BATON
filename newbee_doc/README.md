# TASUKI プログラミング学習ガイド

このフォルダは、プログラミング未経験の方がTASUKIアプリを通じて、実践的なプログラミングを学ぶための教材です。

## 📚 学習の進め方

このフォルダの教材は、**段階的に**読むことで、プログラミングの基礎から実践まで学べるように設計されています。

### 推奨学習順序

```
下記の私の勉強用のmdファイルを作成しています。07から最後まで作成してください。【レベル1: 基礎知識】（所要時間: 2-3時間）
├── 01_プログラミングとは.md
├── 02_TASUKIアプリの全体像.md
└── 03_開発環境の理解.md

【レベル2: 技術要素の理解】（所要時間: 3-4時間）
├── 04_データベースとは.md
├── 05_フロントエンドとバックエンド.md
└── 06_APIとは.md

【レベル3: TASUKI固有の技術】（所要時間: 4-5時間）
├── 07_Flutterとモバイルアプリ.md
├── 08_Supabaseとバックエンド.md
├── 09_AIとGemini.md
└── 10_動画処理とMux.md

【レベル4: 実践】（所要時間: 5-6時間）
├── 11_コードの読み方.md
├── 12_実装の流れ.md

```

**総学習時間**: 約15-18時間（1日2時間で1週間程度）

## 🚀 学習完了後のゴール

このフォルダの教材を全て学習すると、以下ができるようになります：

### ✅ 技術的説明ができる
「TASUKIは、**Flutter**で作られた**モバイルアプリ**で、**Supabase**という**BaaS**を**バックエンド**として使い、**PostgreSQL**で**データベース**を管理しています。動画は**Mux**で処理し、**Gemini AI**が自動でマニュアルを生成します」

### ✅ アーキテクチャを説明できる
「**クライアント（Flutter）**が**HTTP通信**で**Supabase Edge Functions**を呼び出し、それが**Gemini API**にリクエストを送ります。結果は**PostgreSQL**に保存され、**RLS（Row Level Security）**で権限管理されます」

### ✅ 実装の流れを理解できる
「まず**マイグレーション**でテーブルを作り、**RLSポリシー**を設定します。次に**Edge Functions**を実装し、**Flutter**側で**Provider**を使って状態管理します」
